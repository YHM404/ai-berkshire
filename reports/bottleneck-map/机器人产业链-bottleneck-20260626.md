# 机器人产业链瓶颈地图：真正卡脖子的不是“整机”，是关节里的精密件

日期：2026-06-26
研究深度：deep
框架：超级趋势 + 物理瓶颈 + 可交易标的 + 估值安全边际
结论标签：**产业趋势成立；投资上只允许观察，不允许追高**

---

## 0. 一句话结论

机器人产业链是合格超级趋势，但当前二级市场已经把“人形机器人百倍增长”提前计入了多数纯正标的价格。

**产业瓶颈最强的环节：**

1. **精密减速器（谐波/RV）**：S级。物理瓶颈真实，验证周期长，供给集中。
2. **行星滚柱丝杠/高精密丝杠**：A级偏S。人形机器人线性执行器核心件，但上市公司收入兑现仍早。
3. **六维力/力矩传感器**：A级。技术壁垒在标定、解耦、可靠性，市场小但弹性大。
4. **空心杯电机、无框力矩电机、编码器**：B+到A级。重要，但可替代供应商更多，单点瓶颈弱于前三者。

**投资结论：**

- **最像“便宜瓶颈股”的不是人形机器人最纯标的，而是 Nabtesco（6268.T）这类传统工业机器人瓶颈件龙头。**估值约42x PE，10年25x PE退出要求净利润年化增长约5.4%，尚可解释。
- **最纯的谐波减速器标的（绿的谐波、Harmonic Drive Systems）估值已红灯。**产业逻辑强，但安全边际差。
- **柯力传感（603662.SH）是六维力传感器方向最值得放入观察名单的中国上市标的。**估值约62x PE，要求10年净利CAGR约9.5%，不是便宜，但比多数机器人概念股克制。
- **北特科技（603009.SH）和 THK（6481.T）代表滚柱丝杠方向。**产业位置好，但机器人收入尚未充分披露，不能把“扩产规划”当成确定利润。

## 1. 超级趋势验证

### 1.1 已验证底座：工业机器人，不是人形机器人PPT

IFR《World Robotics 2025》显示：

- 2024年全球工业机器人安装量 **542,000台**，连续第4年超过50万台。
- 2024年全球在役工业机器人存量 **4,664,000台**，同比+9%。
- 亚洲占2024年新增部署 **74%**；中国安装 **295,000台**，占全球 **54%**。
- IFR预测2025年全球安装量约 **575,000台**，2028年超过 **700,000台**。
- IFR原文判断：长期增长趋势没有结束迹象。

这说明机器人不是新叙事，而是已有十年级别产业扩张的自动化主线。

### 1.2 高弹性增量：人形机器人，但不能当确定收入

Morgan Stanley《Humanoid 100》把人形机器人价值链拆成 Brain、Body、Integrator，并强调名单仅是“starting point”，不是穷尽清单。其美国TAM模型估算：

- 美国人形机器人年收入：2030年约40亿美元，2040年约2400亿美元，2050年约1万亿美元。
- 中国人形机器人市场：2030年约120亿元人民币，2035年约2160亿元，2050年约6万亿元。

但这些是情景模型，不是订单。报告采用保守处理：**工业机器人是确定性底座，人形机器人是期权，不把远期TAM折现成今天利润。**

### 1.3 Tesla/Optimus 供应链处理原则

Tesla 2025 10-K只确认：公司正在开发并计划商业化AI robots/Bots（含Optimus），但也明确提示：

- Bots仍处于尚未商业化的早期行业。
- 量产爬坡可能延期。
- 部分零部件存在单一供应商风险。
- Bots商业化失败或进展慢于预期，将影响业务前景。

因此，本报告**不采信媒体流传的“某公司进入Optimus供应链”作为确定性订单**。只把这类信息作为观察信号。

## 2. 物理供应链拆解：Layer 0-4

| Layer | 环节 | 内容 | 投资意义 |
|---|---|---|---|
| Layer 0 | 终端需求 | 工厂自动化、物流、仓储、服务机器人、人形机器人 | 判断超级趋势是否成立 |
| Layer 1 | 整机/集成商 | FANUC、ABB、Yaskawa、Tesla、UBTech、Unitree等 | 多数不是最优瓶颈点，竞争与迭代快 |
| Layer 2 | 子系统模块 | 旋转执行器、线性执行器、灵巧手、感知系统、控制系统、电池包 | 重点跟踪，特别是执行器 |
| Layer 3 | 关键零部件 | 谐波/RV减速器、丝杠、六维力传感器、空心杯/无框电机、编码器、轴承 | **核心瓶颈层** |
| Layer 4 | 材料/工艺/设备 | 高精密加工、热处理、齿形设计、应变片、稀土磁材、检测标定设备 | 技术护城河来源，但上市纯标少 |

**瓶颈集中在 Layer 2-3，不在整机。**整机厂会不断换架构，真正难扩的，是精密机械件的一致性、寿命、标定与良率。

## 3. 瓶颈评级

### 3.1 S级：精密减速器（谐波/RV）

**为什么卡：**

- 人形机器人旋转关节需要高扭矩密度、高定位精度、低回差、长寿命。
- 工艺壁垒在材料、齿形、热处理、柔轮疲劳寿命、精密装配和一致性。
- 扩产不是买设备就能解决，验证周期长。
- 工业机器人已有成熟需求，人形机器人只是叠加增量。

**正向证据：**

- Nabtesco公开材料显示，其精密减速器用于中大型工业机器人关节，全球份额约60%，客户包括FANUC、Yaskawa、KUKA、ABB等。
- Harmonic Drive Systems 2025业绩说明会：已有数家公司确认人形机器人量产供货，但2025年仍是“小规模启动”，预计2026年后放量。
- Morgan Stanley把 Harmonic Drive、Nabtesco、LeaderDrive、Shuanghuan 等列为人形机器人“Body / Reducers”关键公司。

**反向证据：**

- 人形机器人短期订单小，2025年大规模量产仍不确定。
- 中国供应商快速追赶，价格战会压低单位利润。
- 最纯标的估值已经透支。

**评级：S（产业瓶颈）/ 投资可买性：黄到红。**

### 3.2 A级偏S：行星滚柱丝杠/高精密丝杠

**为什么卡：**

- 线性执行器要把旋转运动转换成高承载、高效率线性运动。
- 行星滚柱丝杠相比滚珠丝杠承载更高、寿命更好，但加工难度更高。
- Morgan Stanley指出，高端丝杠由欧洲/日本厂商主导，中国公司仍存在效率、载荷、精度差距。

**正向证据：**

- Morgan Stanley把 THK、NSK、SKF、Hiwin、上海北特、江苏恒立列为人形机器人丝杠相关上市公司。
- 北特科技据公开资料规划投资18.5亿元建设人形机器人零部件基地，目标滚柱丝杠量产。
- THK官方财务页面显示其核心产品包括LM Guide、ball screw等直线运动系统；Morgan Stanley提及其SEED线性执行器。

**反向证据：**

- 多数上市公司当前收入仍来自汽车、工业设备或传统直线导轨。
- 人形机器人丝杠收入占比未充分披露。
- 扩产规划不等于订单，更不等于利润。

**评级：A级偏S / 投资可买性：黄灯。**

### 3.3 A级：六维力/力矩传感器

**为什么卡：**

- 人形机器人要安全抓取、行走、接触环境，必须知道力和力矩。
- 六维力传感器难点在应变结构、解耦算法、标定、温漂控制、长期稳定性。
- Morgan Stanley称 ATI（Novanta旗下）是六轴力传感器发明者和市场领导者，中国公司仍在标定与解耦积累上落后。

**正向证据：**

- ATI官网称其六轴力/力矩传感器可为机器人提供实时高精度力控反馈。
- Morgan Stanley将 Novanta、Keli Sensing、Sensata、TE Connectivity列为Force & Torque相关公司。
- 柯力传感已展示机器人六维力传感器，市场开始关注其机器人传感器布局。

**反向证据：**

- 市场规模小，单一公司难靠六维力传感器支撑大市值。
- 国内供应商进入速度快，若人形机器人放量，价格下行会非常快。
- Novanta过于多元，柯力传感机器人收入占比未披露。

**评级：A / 投资可买性：柯力黄灯，Novanta黄红。**

### 3.4 B+：电机、编码器、轴承、磁材

这些都是关键件，但多数不构成强单点瓶颈：

- 无框力矩电机：重要但供应商多。
- 空心杯电机：灵巧手重要，技术门槛较高，但中国厂商进入快。
- 编码器：高精度重要，但可交易纯正标少。
- 轴承：精密度重要，但市场更成熟。
- 稀土磁材：可能受出口管制影响，但机器人不是唯一需求，投资纯正度差。

**评级：B+，作为供应链监控项，不作为主攻方向。**

### 4.2 逐家公司判断

#### Nabtesco（6268.T）——工业机器人确定性最强

**产业位置：**RV减速器龙头。中大型工业机器人关节需要RV减速器，Nabtesco公开材料显示全球份额约60%。
**优点：**已有工业机器人确定需求，不完全依赖人形机器人；客户结构强；估值没有被纯人形机器人概念炒到离谱。
**问题：**人形机器人若更多采用谐波/小型执行器，Nabtesco的人形弹性未必最大；传统工业周期会影响订单。
**结论：**机器人瓶颈股里最值得做进一步基本面深挖的标的。不是最性感，但最接近“事实驱动”。

#### THK（6481.T）——丝杠/直线运动系统龙头，但估值不便宜

**产业位置：**LM Guide、ball screw等直线运动系统核心供应商；Morgan Stanley列为人形机器人线性执行器相关公司。
**优点：**产品能力确定，客户覆盖广，技术积累深。
**问题：**持续经营收入约¥240B，机器人相关收入占比未披露；PE使用正常化口径约86x，安全边际不足。
**结论：**产业地位强，投资动作应等回调或机器人订单进入利润表。

#### Harmonic Drive Systems（6324.T）——纯正，但太贵

**产业位置：**谐波减速器全球核心公司。公司在FY2025/26说明中提到人形机器人客户进入初步量产阶段，并计划扩大北美AI机器人产能。
**优点：**纯正、技术壁垒高、长期受益明确。
**问题：**FY2026净利仅¥1.6B，FY2027指引净利¥4.5B，即使用前瞻利润，估值仍约157x PE；10年25x PE退出仍要求20%+净利CAGR。
**结论：**产业评级S，投资评级红灯。除非订单与利润出现连续超预期，否则不追。

#### 绿的谐波（688017.SH）——中国谐波减速器纯标的，估值最红

**产业位置：**中国谐波减速器核心上市公司，机器人执行器直接相关。
**优点：**业务纯正，国产替代逻辑清晰。
**问题：**市值约¥69.4B，TTM收入约¥613M，PS约113x，PE约507x。10年后以25x PE退出，要求净利年化增长约35%。这不是安全边际，是把极高增长写进股价。
**结论：**产业评级S，投资可买性红灯。只适合观察基本面，不适合用价值投资框架追高。

#### 柯力传感（603662.SH）——六维力传感器方向相对可跟踪

**产业位置：**传统称重/力传感器公司，布局机器人六维力传感器。
**优点：**力传感器技术相关性强；市值约¥18.9B，PE约62x，虽然不便宜，但比纯概念股克制；10年25x PE退出要求约9.5%净利CAGR。
**问题：**机器人六维力传感器收入占比未披露；传统业务仍是主体；市场对机器人弹性可能提前定价。
**结论：**纳入观察名单。关键验证不是新闻，而是机器人传感器收入、毛利率和客户批量订单披露。

#### 北特科技（603009.SH）——滚柱丝杠弹性大，但收入兑现早期

**产业位置：**汽车零部件基础上切入人形机器人滚柱丝杠，规划人形机器人零部件基地。
**优点：**滚柱丝杠方向弹性强；市值仍小于多数大型机器人概念股。
**问题：**PE约127x，10年25x PE退出要求约17.6%净利CAGR；当前收入主体不是机器人丝杠；产能建设到利润释放存在长链条风险。
**结论：**观察，不追。必须等量产、良率、客户、利润率四项确认。

#### Novanta（NOVT）——ATI是好资产，但上市主体不纯

**产业位置：**旗下ATI Industrial Automation为机器人末端工具和六轴力/力矩传感器领导者。
**优点：**产品能力强，客户认可度高，技术壁垒真实。
**问题：**Novanta整体业务包括医疗、精密制造、扫描器等，机器人传感器不是全部；PE约117x，安全边际不足。
**结论：**好公司，但不是便宜瓶颈股。

### 5.2 暂不进入主攻的方向

- **整机厂**：竞争激烈，产品迭代快，硬件成本下降会侵蚀整机毛利。
- **电池/摄像头/通用芯片**：供应链成熟，机器人不是唯一需求，瓶颈纯度不够。
- **稀土磁材**：宏观瓶颈可能存在，但机器人链条纯正度低，需另行作为“稀土出口管制”主题研究。
- **Tesla供应链传闻股**：无官方订单披露前，不进入确定性清单。

## 6. 反面论证：为什么现在不能直接买一篮子机器人瓶颈股

1. **估值已经提前透支。**绿的谐波PS超过100x，Harmonic Drive前瞻PE仍约157x，这要求未来十年持续高增长，容错率极低。
2. **人形机器人商业化时间不确定。**Tesla 10-K已经提示Bot量产与商业化存在延期和失败风险。
3. **客户验证不等于订单。**样品、小批量测试、送样验证，都不能直接折算成营收。
4. **国产替代会伴随价格战。**零部件一旦标准化，供应商扩产速度会很快，毛利率可能先高后低。
5. **机器人细分收入披露不足。**很多公司当前机器人业务占比很小，市场却按机器人公司定价。

芒格式提醒：**“反过来想”：如果十年后这些公司利润没涨10倍，现在的估值靠什么赚钱？**
